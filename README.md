# TOL103M-Assignment-P2
ESP-NOW networking, chat, and ping protocol

**Course:** TÖL103M – Fall 2025  
**Instructors:** Hyytiä and Ágústsson  

## Description
This project implements basic wireless networking functionality using ESP-NOW.

## Implemented Features
- Chat protocol (broadcast and direct messages)
- Ping / pong protocol
- Serial console commands:
  - /ping
  - /date
  - public chat messages
  - direct messages using @nodeid

## Notes
- Packet format follows assignment specification
- CRC-24 used for packet integrity
